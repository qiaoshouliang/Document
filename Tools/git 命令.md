# git 初始化

## git init  
创建 .git

## git add .
添加文件到缓存

## git remote add origin https://gitee.com/qiaoshouliang/41.git
添加远端库的地址

## git pull 
从远程库同步代码到本地库

## git commit
添加文件到本地库

## git push 
将本地库代码同步到远端库




# git 的常用命令

## git stash 讲现有修改缓存下来

## git stash pop 讲缓存的内容pop出来
